# mermaid

Just for the sake of having charts in Markdown files.

```mermaid
flowchart TD
	a[Do you have the time to start yet another pet project?]
	a --> |No| b[Don't do it.]
	a --> |Yes| c[No, you don't, but you should still do it.]
```
